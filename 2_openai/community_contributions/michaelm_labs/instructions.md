# Developer instructions

## Optional Google Auth Dependencies

`google-auth` and `google-auth-oauthlib` are not installed by default. Install them only if you need Google authentication or OAuth flows.

### uv

```bash
uv add google-auth google-auth-oauthlib
```

### pip

```bash
pip install google-auth google-auth-oauthlib
```

### poetry

```
poetry add google-auth google-auth-oauthlib
```
