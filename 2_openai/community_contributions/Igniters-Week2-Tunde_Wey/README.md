## Setup

From the repo root, use the same `.venv` as the course. Ensure `.env` includes `OPENAI_API_KEY`, `SENDGRID_API_KEY`, `SENDGRID_FROM_EMAIL`, and `SENDGRID_TO_EMAIL`.

Open the notebook from Cursor/Jupyter with working directory **repo root** (`agents`) when possible.
