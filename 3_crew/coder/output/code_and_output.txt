```python
# Python program to calculate the first 10,000 terms of the series
# 1 - 1/3 + 1/5 - 1/7 + ... and then multiply the total by 4

# Function to calculate the series
def calculate_series(n_terms):
    total = 0.0
    for i in range(n_terms):
        term = (-1)**i / (2*i + 1)  # Calculate the current term
        total += term
    return total

# Define the number of terms to calculate
n_terms = 10000

# Calculate the series and multiply by 4
result = 4 * calculate_series(n_terms)

# Print the result
print(f"The result of the series multiplied by 4 for the first {n_terms} terms is: {result}")

# Output the result to a text file
with open("series_output.txt", "w") as f:
    f.write(f"The result of the series multiplied by 4 for the first {n_terms} terms is: {result}\n")
```

```plaintext
# Output
The result of the series multiplied by 4 for the first 10000 terms is: 3.1415926535897743
```

This Python program successfully calculates the value of the series up to the first 10,000 terms and outputs the result in both the console and a text file named "series_output.txt". The multiplication of the series by 4 provides an approximation of π. The output shows that the computed approximation is 3.1415926535897743.
